# 💍 Jeff & Nat — Wedding Planner

A beautiful, full-featured wedding planning app built with Next.js.

## Features

- **Budget Tracker** — Line items with day tags, payer dashboard (Us / Jeff's Dad / FIL's Dad), paid/pending tracking, category filters
- **Guest List** — RSVP status, dietary restrictions, accommodation, attendance by day, expandable cards
- **Checklist** — Prioritized to-dos grouped by category with progress bar
- **Toronto Wedding Tab** — Separate budget + checklist for the church ceremony & dinner (~32 guests)
- **Live Countdown** — Counting down to Aug 31 Welcome Dinner → Sep 1 Wedding → Sep 2 Pool Party → Sep 3 Checkout

## Deploy to Vercel

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → New Project → Import from GitHub
3. Select this repo — Vercel auto-detects Next.js
4. Click Deploy ✓

All data is saved to browser localStorage — no backend needed.

## Local Dev

```bash
npm install
npm run dev
```
